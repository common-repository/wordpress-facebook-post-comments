Adds Facebook's Comment Box to all posts except those on the front page

== Description ==

This plugin adds the Facebook Comment Box Javascript code to every blog post and page


== Installation ==

1. Upload `wp-facebook-post-comments` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the desired width for the box and the default number of posts you'd like to appear to the settings (Admin > Settings >Facebook Post Comments )
